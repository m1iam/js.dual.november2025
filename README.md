# Лабораторні роботи з JavaScript

Виконав: Артем Міленко

## Як запустити

Кожна лабораторна знаходиться у своїй папці. Щоб протестувати:

1. Відкрийте HTML файл у браузері (Chrome, Firefox, Safari)
2. Відкрийте консоль розробника (F12 або Cmd+Option+I на Mac)
3. Дивіться результати в консолі

## Структура

- **lab1/** - перша лабораторна
- **lab2/** - завдання з типами даних, циклами
- **lab3/** - масиви, об'єкти, switch
- **lab4/** - методи масивів, фільтрація, пошук
- **lab5/** - замикання (closures)
- **lab6/** - контекст (this, bind, call, apply)

## Що де тестувати

Відкрити: `lab2/index.html`

## Примітки

- У деяких завданнях використовується prompt/alert - це нормально
- Всі результати виводяться в консоль браузера
- Якщо щось не працює - перезавантажте сторінку
